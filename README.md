<h2> NEW EXAM DOMAIN </h2>
<ul>
  <li> Domain 1: Cloud Concepts 24% </li>
  <li> Domain 2: Security and Compliance 30% </li>

  <li> Domain 4: Billing, Pricing, and Support 12% </li>
</ul>
